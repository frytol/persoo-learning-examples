Instructions

  1. Go to Setting - imports and in the list select "products".
  2. Add url https://s3-eu-west-1.amazonaws.com/cz.persoo.images/sampleProductFeeds/snapbacksSample.json to import url and select JSON type
  3. Click to "Status" section and there click "run" import in test environment.
  4. Click the status line to see details. Find "destinations" url to preview and download results.
  5. Note: destinations url contains secret hashes -- set your own in Settings - Credentials - secret hash.
  6. Publish changes to production.
  7. Run import in production (go to Import status, and click "run" in production environment.
