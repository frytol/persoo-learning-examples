Persoo API

Utils

  • Runs custom function when persoo.js loads and processes the command queue: 
    //call function(param1, param2, ..., paramN);
persoo(function, param1, param2,...,paramN);

Notes

The following ways of passing arguments are equivalent: 
persoo('command', param1, param2, {a:1, b:2},{c:3},{d:4,e:5});
persoo('command', param1, param2, {a:1, b:2,c:3,d:4,e:5});
