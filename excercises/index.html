Welcome to persoo demos

How it works

Play yourself with examples given in 'Solutions' or try to implement them in folder 'Excercises'. Download this HTML code to your notebook. Open these html files in your browser to see the result. Edit source files to realize given excercises (simple task, how to implement persoo).

There are two directories:

  • Excercises - we prepared simple excercises, which demonstrates, how persoo works and how you can implement Persoo to your web.
  • Solutions - solved excersices, which you can use as demos, to see how persoo works.

In all examples you can use

  • Persoo documentation
  • Persoo RTP API to make your own request to Persoo REST API
  • jQuery to simplify on() listeners, DOM manipulations,...
  • Twitter Bootstrap for simple and nice design of basic components
  • template rendering lib, i.e. call ejs.render('1 + 2 = <%= 1+2%> and <%= x %>', {x: 'valueX'}); to get strin/HTML code '1 + 2 = 3 and valueX'
  • utility functions, i.e. call // Example: get/set apikey var APIKEY_COOKIE_NAME = 'persooAdminApikey'; var apikey = QueryString.apikey || docCookies.getItem(APIKEY_COOKIE_NAME) || 'qhql07k7dh5h9l7oarrp9tlt'; docCookies.setItem(APIKEY_COOKIE_NAME, apikey);
